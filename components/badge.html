Badge with Background Colors

To set the Background color of the badge, use the color name with a badge-, for example, to use the danger color, use badge-danger. Badges use the standard named colours as defined in the color pallette. The available options are Blue, Gray, Green, Red, Yellow, Cyan. and Black.

Badges can be Blue Gray Green Red Yellow Cyan Black 

                
                  <h3 class="with-badge">Badges can be
                    <span class="badge badge-primary">Blue</span>
                    <span class="badge badge-secondary">Gray</span>
                    <span class="badge badge-success">Green</span>
                    <span class="badge badge-danger">Red</span>
                    <span class="badge badge-warning">Yellow</span>
                    <span class="badge badge-info">Cyan</span>
                    <span class="badge badge-dark">Black</span>
                  </h3>

Badges with Buttons

Badges can be used with buttons by using the with-badge class. Inside the button, the button text is enclosed in a btn-text class and the badge content is enclosed in a badge class. 

                
                  <button class="btn btn-xs with-badge">
                    <span class="btn-text">Button</span>
                    <span class="badge">X</span>
                  </button>
                  <button class="btn btn-s with-badge">
                    <span class="btn-text">Button</span>
                    <span class="badge">X</span>
                  </button>
                  <button class="btn btn-m with-badge">
                    <span class="btn-text">Button</span>
                    <span class="badge">X</span>
                  </button>
                  <button class="btn btn-l with-badge">
                    <span class="btn-text">Button</span>
                    <span class="badge">X</span>
                  </button>
                  <button class="btn btn-xl with-badge">
                    <span class="btn-text">Button</span>
                    <span class="badge">X</span>
                  </button>

Badges with Positioning

Positioned badges use relative positioning at the parent node and absolute positioning to set the position of the badge. They can only be used with buttons using the btn in conjunction with the with-badge and positional class. The badges can be positioned on the top or the bottom by using the top or bottom class. The inner badge text uses the badge and right classes. 

                
                  <button class="btn btn-xl with-badge positional top">
                    <span class="btn-text">Button</span>
                    <span class="badge right">99+</span>
                  </button>
                  <button class="btn btn-xl with-badge positional bottom">
                    <span class="btn-text">Button</span>
                    <span class="badge right">99+</span>
                  </button>
                  <button class="btn btn-s with-badge positional top">
                    <span class="btn-text">Button</span>
                    <span class="badge right">99+</span>
                  </button>
                  <button class="btn btn-s with-badge positional bottom">
                    <span class="btn-text">Button</span>
                    <span class="badge right">99+</span>
                  </button>
