Buttons

Button Colors

Simple Buttons

Buttons are available in primary, secondary, success, warning, danger and info 

                
                  <button class="btn btn-m btn-primary">
                    <div class="btn-text">Primary</div>
                  </button>
                  <button class="btn btn-m btn-secondary">
                    <div class="btn-text">Secondary</div>
                  </button>
                  <button class="btn btn-m btn-success">
                    <div class="btn-text">Success</div>
                  </button>
                  <button class="btn btn-m btn-danger">
                    <div class="btn-text">Danger
                    </div>
                  </button>
                  <button class="btn btn-m btn-warning">
                    <div class="btn-text">Warning</div>
                  </button>
                  <button class="btn btn-m btn-info">
                    <div class="btn-text">Info</div>
                  </button>

Button Text Color Variations

Text in buttons can have four shades, namely: white, gray-light, gray-dark and black 

                
                   <button class="btn btn-m">
                    <div class="btn-text btn-text-white">
                      <a href="#">White Text</a>
                    </div>
                  </button>
                  <button class="btn btn-m">
                    <div class="btn-text btn-text-black">
                      <a href="#">Black Text</a>
                    </div>
                  </button>
                  <button class="btn btn-m">
                    <div class="btn-text btn-text-gray-light">
                      <a href="#">Light Gray Text</a>
                    </div>
                  </button>
                  <button class="btn btn-m">
                    <div class="btn-text btn-text-gray-dark">
                      <a href="#">Dark Gray Text</a>
                    </div>
                  </button>

Link Buttons

                
                   <button class="btn btn-m">
                    <div class="btn-text">
                      <a href="#">Button Text</a>
                    </div>
                  </button>

Outline Buttons

Outline buttons are available in all colors. Use btn-outline instead of btn to use an outline button. 

                
                  <button class="btn-outline btn-m btn-primary">
                    <div class="btn-text">Text</div>
                  </button>
                  <button class="btn-outline btn-m btn-secondary">
                    <div class="btn-text">Text</div>
                  </button>
                  <button class="btn-outline btn-m btn-success">
                    <div class="btn-text">Text</div>
                  </button>
                  <button class="btn-outline btn-m btn-danger">
                    <div class="btn-text">Text</div>
                  </button>
                  <button class="btn-outline btn-m btn-warning">
                    <div class="btn-text">Text</div>
                  </button>
                  <button class="btn-outline btn-m btn-info">
                    <div class="btn-text">Text</div>
                  </button>

Button States

Buttons can be active or disabled. Use the disabled class to make a button disabled. 

                
                  <button class="btn btn-m btn-primary disabled">
                    <div class="btn-text">Disabled</div>
                  </button>
                  <button class="btn btn-m btn-secondary disabled">
                    <div class="btn-text">Disabled</div>
                  </button>
                  <button class="btn btn-m btn-success disabled">
                    <div class="btn-text">Disabled</div>
                  </button>
                  <button class="btn btn-m btn-danger disabled">
                    <div class="btn-text">Disabled</div>
                  </button>
                  <button class="btn btn-m btn-warning disabled">
                    <div class="btn-text">Disabled</div>
                  </button>
                  <button class="btn btn-m btn-info disabled">
                    <div class="btn-text">Disabled</div>
                  </button>

Button Sizes

Buttons can be of xs, s, m, l and xl sizes. 

                
                  <button class="btn btn-xs">
                    <span class="btn-text">XS Button</span>
                  </button>
                  <button class="btn btn-s">
                    <span class="btn-text">S Button</span>
                  </button>
                  <button class="btn btn-m">
                    <span class="btn-text">M Button</span>
                  </button>
                  <button class="btn btn-l">
                    <span class="btn-text">L Button</span>
                  </button>
                  <button class="btn btn-xl">
                    <span class="btn-text">XL Button</span>
                  </button>
