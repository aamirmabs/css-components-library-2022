Product Cards

Cards with dismiss

Any parent element that has the with-dismiss class can be deleted from the DOM by placing a element with the dismiss-trigger class. Internally, the dismiss-trigger element will have an event listener attached to it that looks for the closest parent containing with-dismiss and deletes it from the DOM.
