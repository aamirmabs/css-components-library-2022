Avatars with Status

Avatars can have a status if needed. The colors names used are: primary, secondary, success, danger, warning, info, light and dark. The size of the status highlighter will alter based on the size of the avatar size (.avatar-l, .avatar-m, etc.) class.

Avatars Groups

Avatars can be grouped either horizontally or vertical. All the avatars have to be enclosed in a .avatar-group class. Internally flex is used to set the alignment and the default direction of a group is horizontal/row-direction. To make the .avatar-group align vertically, use the .vertical class in conjunction.

Horizontal Avatar Group

                              
                                <div class="avatar-group horizontal">
                                  <div class="avatar avatar-l">
